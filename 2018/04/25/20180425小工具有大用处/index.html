小工具有大用处，编程道路上从此少点烦恼

刘宇佳 发表于 2018-04-25

这篇写的基本上算是个半技术文章，这些电脑上的小工具都是我用过觉得超级赞的，推荐给你，希望这些小东西能给你带来惊喜，增加你的学习和办事效率。

开始学习编程以来发现了很多有意思的玩意儿，很多时候用到的都是一些特别小众的软件和工具，但是其丰富的功能以及多元的接口总是能实现很多的需求，单单是这些不起眼的玩意儿就足以见得这个行业的从业者们有多么朝气蓬勃！

###JSFiddle

随着网页应用的越来越广泛，学习前端的人是越来越多了，曾经被很多的后端工程师们瞧不上的前端技术现在在各个地方都能用上，一部分后端工程师甚至逐渐转向了前端开发的工作，简直是实力打脸啊！

现在学习前端技术甚至不用编译器了，一个简单的网页就可以满足前端配置环境的需求，上面图片中就是这个网站的编译呈现效果。三个框内同时编程，显示效果直接在第四个框中呈现，有了这个网站，你甚至不用买外接屏幕了。

###Visual Studio Code

记得曾经学编程的时候装Visual Studio软件，下载用了两三个小时，安装用了半个多小时，结果装上之后因为电脑的配置比较低，Visual Studio运行起来特别卡，只好删了换别的编译器。

因为这个笨重的编译器，微软不知道被骂了多少次，都快被唾沫淹没了。不过微软公司还是知耻而后勇，做了Visual Studio Code这样牛的软件。

Visual Studio Code很轻，花几分钟下载，再花几分钟安装就可以用了，占内存也不小，跑起来也快。而且不知道哪来那么多的开发者，给Visual Studio Code提供了各种各样的开发组件，几乎所有的编程语言都能在Visual Studio Code上跑了。就连微信小程序的开发工具在上面都有组件。微软公司还是有牛人的，Visual Studio Code真是一款伟大的产品。

###TotalFinder

这个工具即除了程序员必备之外，一般用Mac的人都值得拥有，因为它是提升工作效率的利器。

苹果公司的Finder实在是太烂了，文件移来移去太不方面。TotalFinder集成了MacOS和Windows中文件处理的优势，打造除了超级高效的应用，只要花十几美元就能得到它的永久使用权。只要你用上一次，保证你以后也停不下来！
