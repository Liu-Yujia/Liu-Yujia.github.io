程序员的健康问题可能也是屏幕前的你所要注意的问题

刘宇佳 发表于 2018-01-14

程序员的健康问题，可能也是屏幕前的你所要注意的问题

你能想象到一个退伍的军人学习编程的场景么，同时这位军人还满世界跑去学习武术，还抽空玩儿乐器，还把自己学编程的经历写成了一本书——《Learn Python The Hard Way》，我天，简直太传奇。

典型的斜杠青年，普通人在一生中只要做好上面的一样事情就会很了不起，可人家同时把几件事情都做到很不错的水准。

就拿我来说，在上高中的时候跟风学过一阵子吉他，后来死记硬背地学会了一首歌以后再也没碰过吉他；后来在上大学时还上过计算机程序设计的课程，结果到现在还没把编程给学会；更要命的是，我很多休闲的时间都没用来锻炼健身，身体一直处于亚健康状态。

这些我从小到大一直有机会做好但无一例外都没做好的事情，到人家那儿反倒相当的轻松和写意。要不是最近我因为久坐出现了肩部酸疼的症状，我可能都没机会读到这个人的文章，更不可能知道这个人。

当然今天的主题不是程序，不是跨界，而是现在坐在办公桌和电脑前的大家普遍要面对的健康问题。

我之前在上学的时候还没什么感觉，每天及时有久坐的情况，可能单次的久坐顶多就是两个多小时。上班以后一直对这个事情也没太在意，知道最近肩膀酸疼才反应多来，我每天要有两个时间段久坐持续三个小时，而且大部分的时间还是对着电脑。

尽管我的本职工作不是程序员，但是在这样的工作环境和工作强度下，身体的消耗跟程序员可能没什么区别。相信很多人目前跟我也一样，因无知或无意识，一直默默消耗自己的健康。

这个叫做Zed Shaws，是个美国的退伍军人，在做计算机程序设计工作的时候，他发现了很多程序员们平日的职业病：

• 腕部疼痛（重复性劳损）
• 盯着看移动的打印进度以及其后的句点造成的眼部问题
• 姿势不当造成的背部不适，尤其是背下侧，肩上侧。
• 肠道和尿道的问题（不能正常大小便）
• 摄入太多咖啡因，而且喝水少造成的脱水
• 坐姿保持太久，造成男性的痔疮或前列腺问题
• 光照不足造成维生素D缺乏
• 熬夜或者喝咖啡过多导致的睡眠失调
• 缺乏伸展导致普遍的僵硬和疼痛

看到这儿是不是有很多人已经有点感同身受了？

上面的这些职业病条款里，有些条款甚至是我完全想不到的，比如肠道和尿道的问题（不能正常大小便）；坐姿保持太久，造成男性的痔疮或前列腺问题；光照不足造成维生素D缺乏。可能我这么久坐十年八年，这些症状才会慢慢地浮现出来，所以在事发之间能看到这篇文章真得还挺令我欣慰的。

另外一些症状可能是上班之后的人经常能感受到的，甚至超级多的人都因此痛苦不堪。

腕部疼痛，应该是很多的程序员们都在经历的痛苦，我身边的同学做程序员的，都在说他们的腕部经常会疼痛。他们中有的人去了医院拍片诊断，被确认得了腱鞘炎。

眼部问题可能是所有新生一代的人要面临的问题，被各种屏幕占据了日常生活的今天，几乎找不到不近视的人。当然更加痛苦的就是总盯着光标看，视线一直停留在一个闪烁的东西上，这样伤眼镜伤得更厉害。

“姿势不当造成的背部不适，尤其是背下侧，肩上侧”，这点可能是只要久坐的人都会面临的问题，从小的时候就听大人们说身边有很多得“腰椎间盘突出”的人，平时坐也不是站也不是，还不好治。我在14年考研前备考的时候也出现过这样的情况，当时每天要在书桌前坐好几个消失复习，有段时间一度腰疼到没办法复习考试内容，于是每天二三十个仰卧起坐，早上早起半小时去操场跑两圈，坚持了大概十几天，疼痛终于算是消失了。

关于这些疾病和症状，在Zed Shaws都有简单有实用的防止和解决方法，大家可以点开链接看，如果现在已经就有很严重症状的，还是赶紧去医院看看。

愿所有人在高效工作的同时能保持健康的身体！
