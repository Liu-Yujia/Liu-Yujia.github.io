YC创业课的思考一

刘宇佳 发表于 2018-01-14

YC创业课是硅谷的创业大牛们给斯坦福大学的本科生们开设的课程，我们不得不惊叹斯坦福大学的精英们是多么幸福。

前段时间买书的时候顺便捎着买了一本李笑来的《斯坦福大学创业成长课》。这是李笑来老师在看了斯坦福大学的YC创业成长课程《如何开始创业》（How To Start A Startup），之后的一些感悟。

人和人的差别就是不一样，人家在网上看个英文的视频课程，然后写点感悟就能出书卖钱。这个课程其实是在网上免费公开的，但是我没法看，因为英语的听读能力实在太差了，只有吃别人嚼过的二手知识来获取一些高质量的知识。

其实这种获取知识的体验感超级差，英文世界中其实有很多的专业名词以及新鲜的信息需要通过英文理解，再看别人的翻译之后总觉得知识的学习是条断头路，这种局限感其实很讨厌。不过好在YC本来的内容足够的丰富精彩，我也通过二手知识获得了很多思考。

YC创业课是硅谷的创业大牛们给斯坦福大学的本科生们开设的课程，我们不得不惊叹斯坦福大学的精英们是多么幸福。更令我惊叹的是，这些牛人把所有精心准备的内容都公开出来，让全世界的人都无偿地享受他们的知识。

翻开这本书读了几篇之后，我发现跟我预想的不太一样。这本书中讲的更多的是个人思维方式以及思维陋习的纠正，而不是大牛们传奇的创业故事。他们几乎把最干的干货呈献给大家了。

今天感触最深的是一篇纠正逻辑错误的文章《一个70%的人都会犯的逻辑错误》：

认真读完文章的观点和例子之后，我细想了一下自己平时所坚持的那些论断。发现那些论断都是错的，都错在了这个70%所犯的逻辑错误上。

这个逻辑错误是“肯定后见”。

If P then Q；
Q，therefore P.

这是个极其不容易被发现的陷阱，细想了一下，不光是自己的逻辑经常会中枪；平时人们举的反例都是这种有漏洞的逻辑小套路。

就拿我经常和父亲正面的美国言论自由为例：

美国人经常标榜自己言论自由，然后我也总是试图找一些美国言论自由的证据，有一次，我通过网络看奥巴马总统的演讲，发现在总统演讲的时候竟然有人可以高声喧哗打断总统的演讲。因此我就下了一个结论。

If 美国社会有言论自由的风气，then人们在可以骂总统。
人们不可以公开质疑总统，therefore 这个社会就没有言论自由。
中国人们不可以公开质疑国家主席，therefore 中国就没有言论自由。

先抛开本身命题的正确性不说，我的这种习惯性的推理方式就很明显犯了“肯定后见”的逻辑错误。判断言论自由的维度很多，根本不是一个方面就能决定的，现在中国很多方面都可以随意发言和讨论，没有比现在更言论自由的时候了。

我之前还一直因为这个逻辑说服了我爸沾沾自喜，其实这种逻辑错误最可怕。自己把自己骗了，还根本不知道。

还有很多时候，我们不知不觉被别人的反向质疑给骗了：

比如说我们经常要考某个高级别的证书或学习某项特别难的技能时候会受到周围很多人的善意的阻止，因为我们要做的事情是别人尝试之后并没有做成功的事情，所以他们给的建议是你尽量不要开始，反正是失败，何必浪费时间。

If周围的人都没有考过，then这个证书很难考
这个证书很难考，therefore 你也考不过。

首先他们没有调差过足够多的样本，另外他们在了解样本的过程中，并没有考察样本在考试的时候花了多少时间。

通常情况下，人们总是以没有证实和没有统计依据的经验来质疑自己的行动。我们要经常动用自己的思维来判断这些质疑到底可不可信。

以我的经验，很多的反向质疑都是站不住脚的。

关于如何避开这种逻辑陷阱，李笑来老师给出的自己的方式很有用处，在遇到事情的时候经常要问自己：“反过来不一定成立吧”，然后多想想一些反面的例子，这样就有可能让自己避开很多因逻辑错误而造成的严重后果。
